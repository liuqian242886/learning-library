HOW IT WORKS
如何添加新课程
1

提供课程入口

粘贴课程 URL，或直接说课程名称和平台。支持 DeepLearning.AI、YouTube、Bilibili、Coursera。

2

登录对应平台

部分平台（DeepLearning.AI、Coursera）需要在浏览器中保持登录状态，字幕抓取依赖认证 session。

3

自动提取字幕

系统识别平台类型，自动调用对应 API 抓取所有课时字幕。无法抓取时提供手动上传 .srt/.vtt 的备用方案。

4

生成中文笔记

基于完整字幕内容，生成带思维导图、课时标签、词汇表的中文 H5 页面，并自动更新这里的索引。
